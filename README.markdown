My Blog - Its modified Blogcast:
======================================================
* fixed migrations
* hamlized
* Tag support
* updated WDM
* Twitter Support
* Added social links
* Using Sitemap Generator Gem

Install
======================================================
* bundle install
* run migrations
* change settings in config/initializers/configuration.rb
* rake db:seed
* rails server
* login at : /admin
* After creating a post, run crontask or exec rake task "sitemap:refresh"
